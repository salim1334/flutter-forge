# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-23

### Added

- `create-flutter-forge` and `flutter-forge` CLI binaries
- `create` command — `flutter create` + full template overlay
- `feature` command — scaffold feature module + auto-wire routes
- Generated Flutter app with GetX, named routes, SQLite, `.env` config
- Sample features: splash, onboarding, shell, home, explore, settings
- Strict `analysis_options.yaml` with `flutter_lints`
- Documentation in `docs/` and generated `FEATURES.md`
